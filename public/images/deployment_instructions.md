# Инструкция по развертыванию Laravel-проекта на хостинге ps.kz

Этот план поможет вам безопасно и правильно опубликовать ваш сайт.

---

### Шаг 1: Подготовка файлов проекта

1.  **Упакуйте проект в ZIP-архив.** На вашем компьютере найдите папку с вашим проектом (где лежат папки `app`, `resources`, `public` и т.д.). Выделите все файлы и папки и создайте из них один ZIP-архив. Назовите его, например, `fort-inform-site.zip`.

---

### Шаг 2: Загрузка файлов на хостинг

1.  **Войдите в cPanel** на ps.kz.
2.  Откройте **"Менеджер файлов"**.
3.  Перейдите в корневую папку вашего сайта, которая обычно называется `public_html`.
4.  Нажмите кнопку **"Отправить"** (Upload) и выберите ваш ZIP-архив `fort-inform-site.zip`.
5.  После завершения загрузки вернитесь в "Менеджер файлов", найдите ваш архив, кликните на него правой кнопкой мыши и выберите **"Извлечь"** (Extract).

---

### Шаг 3: Настройка базы данных

1.  В cPanel найдите раздел **"Базы данных"** и откройте **"Мастер баз данных MySQL®"**.
2.  **Создание базы данных:**
    *   Придумайте имя для новой базы данных (например, `fortinform_db`). Запишите полное имя (оно будет выглядеть как `вашлогин_fortinform_db`).
3.  **Создание пользователя:**
    *   Придумайте имя пользователя (например, `user_db`) и сгенерируйте надежный пароль. **Обязательно сохраните имя пользователя и пароль!**
4.  **Добавление прав:**
    *   На последнем шаге мастера поставьте галочку **"Все права"** (All Privileges) и нажмите "Следующий шаг".

---

### Шаг 4: Настройка конфигурации Laravel (`.env` файл)

1.  В "Менеджере файлов" (`public_html`) найдите файл `.env.example`.
2.  **Переименуйте** его в `.env`.
3.  Откройте файл `.env` для редактирования.
4.  Найдите и **отредактируйте** следующие секции:

    **Подключение к базе данных:**
    ```ini
    DB_CONNECTION=mysql
    DB_HOST=127.0.0.1
    DB_PORT=3306
    DB_DATABASE=полное_имя_базы_данных_из_шага_3
    DB_USERNAME=полное_имя_пользователя_из_шага_3
    DB_PASSWORD=пароль_пользователя_из_шага_3
    ```

    **Настройка почты (Gmail):**
    ```ini
    MAIL_MAILER=smtp
    MAIL_HOST=smtp.gmail.com
    MAIL_PORT=587
    MAIL_USERNAME=fortinform@gmail.com
    MAIL_PASSWORD=ВАШ_ПАРОЛЬ_ПРИЛОЖЕНИЯ_ОТ_GMAIL
    MAIL_ENCRYPTION=tls
    MAIL_FROM_ADDRESS="fortinform@gmail.com"
    MAIL_FROM_NAME="${APP_NAME}"
    ```
    > **Важно:** `MAIL_PASSWORD` — это не ваш обычный пароль, а специальный **Пароль приложения**. Его можно сгенерировать в настройках безопасности вашего аккаунта Google (требуется включенная двухэтапная аутентификация).

5.  **Сохраните** изменения в файле `.env`.

---

### Шаг 5: Выполнение миграций (Создание таблиц)

1.  В cPanel найдите и откройте **"Терминал"**.
2.  Перейдите в папку вашего сайта командой:
    ```bash
    cd public_html
    ```
3.  Запустите миграции для создания таблиц в базе данных:
    ```bash
    php artisan migrate
    ```

---

### Шаг 6: Финальная настройка веб-сервера

1.  В cPanel в разделе "Домены" откройте **"Домены"**.
2.  Найдите ваш домен и измените его **"Корневой каталог документов"** (Document Root) на `public_html/public`.
3.  Нажмите **"Сохранить"**.

**Готово!** Ваш сайт должен быть полностью рабочим.
